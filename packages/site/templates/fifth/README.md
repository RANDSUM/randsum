# @randsum/fifth - Live Example

This is a live code example for the @randsum/fifth package.

## Try it out

Edit the code in `index.ts` to experiment with D&D 5th Edition rolls!

## Examples

- `roll({ modifier: 5 })` - Basic roll with modifier
- `roll({ modifier: 5, rollingWith: 'Advantage' })` - Roll with advantage
- `roll({ modifier: -2, rollingWith: 'Disadvantage' })` - Roll with disadvantage
