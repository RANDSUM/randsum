# @randsum/root-rpg - Live Example

This is a live code example for the @randsum/root-rpg package.

## Try it out

Edit the code in `index.ts` to experiment with Root RPG rolls!

## Examples

- `rollRootRpg(0)` - Roll with no modifier
- `rollRootRpg(2)` - Roll with +2 modifier
- `rollRootRpg(-1)` - Roll with -1 modifier

## Outcomes

- **Strong Hit**: 10 or higher
- **Weak Hit**: 7-9
- **Miss**: 6 or lower
