# @randsum/roller - Live Example

This is a live code example for the @randsum/roller package.

## Try it out

Edit the code in `index.ts` to experiment with different dice rolls!

## Examples

- `roll('1d20')` - Roll a single 20-sided die
- `roll('4d6L')` - Roll 4d6, drop lowest (D&D ability scores)
- `roll('2d20H')` - Roll 2d20, keep highest (advantage)
- `roll('2d20L')` - Roll 2d20, keep lowest (disadvantage)
