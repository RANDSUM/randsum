# @randsum/salvageunion - Live Example

This is a live code example for the @randsum/salvageunion package.

## Try it out

Edit the code in `index.ts` to experiment with Salvage Union table rolls!

## Examples

- `rollTable()` - Roll with default table
- `rollTable('Morale')` - Roll with specific table
- `rollTable('Core Mechanic')` - Roll with core mechanic table

## Hit Types

- **Nailed It**: 20
- **Success**: 11-19
- **Tough Choice**: 6-10
