# RANDSUM Documentation

This directory contains the documentation for the RANDSUM library.

## Structure

- `index.html` - Landing page for the documentation
- `api/` - API documentation generated by TypeDoc (generated during CI)

## Generating Documentation Locally

To generate the API documentation locally, run:

```bash
# Install dependencies
bun install

# Build packages
bun run build

# Generate documentation
bun run typedoc --entryPointStrategy packages --out docs/api packages/*/src
```

## Viewing Documentation Locally

After generating the documentation, you can view it by opening `docs/index.html` in your browser, or by using a local server:

```bash
# Using Python's built-in HTTP server
python -m http.server -d docs

# Or using a Node.js server like serve
npx serve docs
```

## Documentation Deployment

The documentation is automatically deployed to GitHub Pages when changes are pushed to the main branch and CI passes successfully.

The live documentation can be accessed at: [https://randsum.github.io/randsum/](https://randsum.github.io/randsum/)
