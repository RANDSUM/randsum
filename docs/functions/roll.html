    • roll(...args: NumericRollArgument[]): NumericRollResult

      Rolls dice based on the provided arguments.

      Parameters

      • ...args: NumericRollArgument[]

        One or more roll arguments. Can be numbers, notation strings, or option objects

      Returns NumericRollResult

      A RollResult containing the results of the roll(s)

      Remarks

      This function supports multiple formats for specifying dice rolls:

      • Standard notation strings (e.g., "4d6", "2d20H")
      • Numbers (e.g., 20 for 1d20)
      • Option objects for detailed configuration

      Example

      // Basic rolls
      roll(20) // Roll 1d20
      roll("4d6") // Roll 4d6

      // Advanced notation
      roll("4d6L") // 4d6, drop lowest
      roll("2d20H") // 2d20, keep highest
      roll("4d6R{<3}") // 4d6, reroll values below 3

      // Using options object
      roll({
        quantity: 4,
        sides: 6,
        modifiers: {
          drop: { lowest: true },
          reroll: { below: 3 }
        }
      })

      // Multiple dice pools
      roll("2d20", "4d6") // Roll both dice pools

      Throws

      If the dice notation is invalid

      Throws

      If the number of sides is less than 1

      Throws

      If the quantity is less than 1
