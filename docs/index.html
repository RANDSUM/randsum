Packages

  • @randsum/core - Core utilities and types (internal package)
  • @randsum/dice - Main dice rolling implementation
  • @randsum/notation - Dice notation parser
  • @randsum/5e - D&D 5th Edition compatible dice rolling
  • @randsum/root-rpg - ROOT RPG compatible dice rolling

Getting Started

Install the main dice package:

npm install @randsum/dice
# or
yarn add @randsum/dice
# or
bun add @randsum/dice

Basic usage:

import { D20, roll } from '@randsum/dice';

// Roll a D20
const result = D20.roll();
console.log(`You rolled: ${result}`);

// Roll with advantage (2d20, drop lowest)
const advantageRoll = roll('2d20L');
console.log(`Advantage roll: ${advantageRoll.total}`);
