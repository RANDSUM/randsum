    Function roll

    • roll<T = string>(...args: RollArgument<T>[]): RollerRollResult<T>

      Rolls dice according to RANDSUM notation.

      Accepts multiple roll arguments and an optional configuration object. Each argument can be a dice notation string, a number (sides), or a roll options object.

      Never throws - errors are returned in the result's error property.

      Type Parameters

      • T = string

        Type for custom dice faces. Defaults to string.

        • For standard numeric dice (notation strings or numbers), result contains string representations of the roll values (e.g., ["5", "3", "6"]).
        • For custom faces (options with sides: T[]), result contains the actual face values of type T.

      Parameters

      • ...args: RollArgument<T>[]

        One or more roll arguments (notation strings, numbers, or options objects), optionally followed by a RollConfig object

      Returns RollerRollResult<T>

      Roll result containing individual rolls, total, result array, and error (if any)

      Example: Number (1 die, sides = number) 

      const result = roll(20) // 1d20

      Example: Notation string 

      const result = roll("2d6")
      result.total // => Sum of 2d6
      result.result // => ["3", "5"] - string representations

      Example: Options object 

      const result = roll({ sides: 6, quantity: 4, modifiers: { drop: { lowest: 1 } } }) // same as 4d6L

      Example: Custom faces (Fate dice) 

      const result = roll({ sides: ['+', '+', ' ', ' ', '-', '-'], quantity: 4 })
      result.result // => ["+", "-", " ", "+"] - actual face values

      Example: With custom RNG 

      const seeded = createSeededRandom(42)
      const result = roll("1d20", { randomFn: seeded })

      Example: Multiple rolls 

      const result = roll("1d20+5", "2d6+3")
      // Returns combined total of both rolls

      Example: Error handling 

      const result = roll("invalid notation")
      if (result.error) {
        result.error.message // => "Invalid dice notation: ..."
      }
