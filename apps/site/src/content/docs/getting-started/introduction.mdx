RANDSUM is an open-source, TypeScript-first [dice notation](/reference/dice-notation/) ecosystem for tabletop RPGs and game development. Throw dice, not exceptions.

## Why RANDSUM?

- **Zero dependencies** -- the core roller has no runtime dependencies
- **Full TypeScript support** -- strict types, branded notation strings, and exhaustive result types
- **Throws on invalid input** -- hardcoded notation never throws in practice; validate user-provided strings with `validateNotation()` or `isDiceNotation()`
- **Modular architecture** -- install only the packages you need
- **Game-specific packages** -- first-class support for [D&D 5e, Blades in the Dark, PbtA, and more](/games/overview/)

## How it works

At its simplest, RANDSUM lets you roll dice with a single function call:

<CodeExample code={`import { roll } from '@randsum/roller'

// Roll a d20
const result = roll(20)
console.log(result.total) // 1-20

// Roll 4d6, drop the lowest
const abilityScore = roll('4d6L')

// Roll with advantage
const attack = roll('2d20H+5')`} />

<RollerPlayground defaultNotation="1d20" client:only="react" />

The `roll()` function accepts numbers, notation strings, or options objects -- and you can mix and match them freely.

## The ecosystem

RANDSUM is organized as a monorepo with several packages:

| Package | Description |
|---|---|
| `@randsum/roller` | Core dice engine -- notation parsing, modifiers, roll execution |
| `@randsum/games/fifth` | D&D 5th Edition mechanics |
| `@randsum/games/blades` | Blades in the Dark action rolls |
| `@randsum/games/pbta` | Powered by the Apocalypse (2d6+stat) |
| `@randsum/games/daggerheart` | Daggerheart hope/fear dice |
| `@randsum/games/root-rpg` | Root RPG mechanics |
| `@randsum/games/salvageunion` | Salvage Union table rolls |

Every game package depends only on `@randsum/roller` -- they never depend on each other.
