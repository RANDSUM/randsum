`@randsum/cli` is a terminal dice roller with two modes: instant roll-and-exit for quick results, and an interactive TUI for exploration.

## Quick Roll

Pass notation as arguments for an instant result:

<CodeExample lang="bash" code={`randsum 4d6L          # Roll 4d6, drop lowest (D&D ability score)
randsum 2d20L+7       # Advantage attack roll with +7
randsum 1d20 -v       # Verbose output with full breakdown
randsum 3d6 --json    # JSON output for scripting`} />

## Interactive Mode

Run with no arguments (or `-i`) to open the interactive terminal UI:

<CodeExample lang="bash" code={`randsum
randsum -i`} />

The TUI provides:
- **Live notation input** with color-coded token highlighting
- **Modifier reference** with keyboard navigation — browse all modifiers and add them to your notation
- **Roll result panel** showing individual dice, applied modifiers, and final total
- **Token focus navigation** — arrow keys move between notation tokens; left from position 0 enters roll focus

## Flags

| Flag | Description |
|---|---|
| `-i, --interactive` | Launch interactive TUI mode |
| `-v, --verbose` | Show detailed roll breakdown |
| `--json` | Output results as JSON |
| `-r, --repeat N` | Roll N times |
| `-s, --seed N` | Use seeded random for reproducible rolls |
| `-V, --version` | Show version |
| `-h, --help` | Show help |

## Examples

<CodeExample lang="bash" code={`# D&D character generation (6 ability scores)

# Attack roll with advantage, verbose

# Blades in the Dark pool

# JSON output for scripts
randsum 2d6+3 --json

# Seeded roll (reproducible result)
randsum 1d20 -s 42`} />

## Dice Notation

Uses [RANDSUM Dice Notation](/reference/dice-notation/) — the same notation accepted by `@randsum/roller`.
