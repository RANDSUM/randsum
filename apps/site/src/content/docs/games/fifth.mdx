The `@randsum/fifth` package provides [Dungeons & Dragons 5th Edition](https://www.dndbeyond.com/) mechanics including ability checks, saving throws, and attack rolls with full support for advantage and disadvantage.

## The d20 System

In Dungeons & Dragons 5th Edition, most action resolution uses a single twenty-sided die (d20). You roll the die, add an ability modifier (typically −4 to +10), and compare the result against a target number called a Difficulty Class. Rolling with **advantage** means rolling two d20s and keeping the higher result — mechanically reflecting favorable circumstances like a spell, high ground, or helping ally. **Disadvantage** is the reverse: roll two d20s and keep the lower. The `@randsum/fifth` package implements these mechanics exactly, handling the 2d20 rolls, modifier application, and advantage/disadvantage resolution with full TypeScript types.

  rollingWith: { disadvantage: true }

## API

### `roll(arg: FifthRollArgument)`

**Input:**

| Property | Type | Description |
|---|---|---|
| `modifier` | `number` | Ability/skill modifier (-30 to +30) |
| `rollingWith` | `FifthAdvantageDisadvantage \| undefined` | Optional `{ advantage: true }` or `{ disadvantage: true }` |

**Returns:** `GameRollResult` with:

| Property | Type | Description |
|---|---|---|
| `result` | `number` | Final total (d20 + modifier) |
| `total` | `number` | Same as result |
| `rolls` | `RollRecord[]` | Raw dice data from the core roller |

## Outcomes

- **Normal roll**: 1d20 + modifier
- **Advantage** (`{ advantage: true }`): Roll 2d20, drop lowest, + modifier (equivalent to `2d20L+modifier`)
- **Disadvantage** (`{ disadvantage: true }`): Roll 2d20, drop highest, + modifier (equivalent to `2d20H+modifier`)

The modifier is validated to be a finite number in the range -30 to +30.

## Common patterns

<CodeExample code={`import { roll } from '@randsum/fifth'

// Attack roll with +5 to hit
roll({ modifier: 5 })

// Attack with advantage
roll({ modifier: 5, rollingWith: { advantage: true } })

// Saving throw with -1 penalty and disadvantage
roll({ modifier: -1, rollingWith: { disadvantage: true } })

// Perception check with +3 Wisdom
roll({ modifier: 3 })`} />

:::tip
For D&D ability score generation (4d6 drop lowest), use the core roller directly: `roll('4d6L')` from `@randsum/roller`.
:::
