The `@randsum/root-rpg` package provides mechanics for [Root RPG](https://magpiegames.com/pages/root-rpg), a tabletop roleplaying game set in the world of the Root board game. It uses a 2d6 + bonus system with success thresholds similar to Powered by the Apocalypse games.

## Overview

- Rolls 2d6 + stat bonus
- 10+ = Strong Hit (complete success)
- 7-9 = Weak Hit (partial success, success with cost)
- 6- = Miss (failure)
- Simple API: pass a single bonus number

## API reference

### `roll(bonus: number)`

**Input:**

| Parameter | Type | Description |
|---|---|---|
| `bonus` | `number` | Stat modifier (-20 to +20) |

**Returns:** `GameRollResult` with:

| Property | Type | Description |
|---|---|---|
| `result` | `RootRpgResult` | `'Strong Hit' \| 'Weak Hit' \| 'Miss'` |
| `total` | `number` | Sum of 2d6 + bonus |
| `rolls` | `RollRecord[]` | Raw dice data from the core roller |

## Result interpretation

| Outcome | Total | Description |
|---|---|---|
| `Strong Hit` | 10+ | Complete success -- you achieve your goal |
| `Weak Hit` | 7-9 | Partial success -- you do it, but with a cost or complication |
| `Miss` | 6- | Failure -- things go badly |
