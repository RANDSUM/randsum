RANDSUM supports six tabletop RPG systems, each with its own dice mechanics and outcome types. Use the comparison below to find the right package for your game.

Every game package wraps the core `@randsum/roller` engine, so they all share the same reliable dice rolling foundation. The difference is in how each package validates input, configures the roll, and interprets the result.

## Package Comparison

| Package | Game System | Dice | Input | Outcomes |
|---|---|---|---|---|
| `@randsum/games/fifth` | D&D 5th Edition | 1d20 + modifier | `{ modifier, rollingWith }` | numeric total + roll detail |
| `@randsum/games/blades` | Blades in the Dark | 0-10d6 dice pool | dice count (0-10) | critical / success / partial / failure |
| `@randsum/games/pbta` | Powered by the Apocalypse | 2d6 + stat | `{ stat, forward, ongoing, advantage, disadvantage }` | strong_hit / weak_hit / miss |
| `@randsum/games/daggerheart` | Daggerheart | 2d12 (Hope + Fear) | `{ modifier, rollingWith }` | Hope/Fear total, critical, with fear |
| `@randsum/games/root-rpg` | Root RPG | 2d6 + bonus | bonus number | Strong Hit / Weak Hit / Miss |
| `@randsum/games/salvageunion` | Salvage Union | 1d20 | table name (optional) | Nailed It / Success / Tough Choice / Failure / Cascade Failure |

All packages wrap `@randsum/roller` and are available on npm under the `@randsum` organization.

## Choosing a System

If you are building a D&D 5e tool — character sheets, encounter trackers, spell attack calculators — reach for `@randsum/games/fifth`. It handles the d20+modifier system natively, including advantage (roll 2d20, keep highest) and disadvantage (keep lowest), and returns both the numeric total and which die was used.

For Blades in the Dark, use `@randsum/games/blades`. The game uses a dice pool where only the single highest die matters: 6 is a full success, 4–5 is a partial success, 1–3 is a failure, and two or more 6s is a critical. Pass a dice count (0–10) and get one of four outcome tiers back.

If you are targeting any Powered by the Apocalypse game — Dungeon World, Monster of the Week, Apocalypse World, Masks, and others — use `@randsum/games/pbta`. The 2d6+stat engine is universal across the whole family: 10+ strong hit, 7–9 weak hit, 6− miss.

For systems not covered by a dedicated package, use `@randsum/roller` directly with notation strings or options objects. The full modifier system (drop, keep, reroll, explode, cap, and more) handles most custom mechanics without additional abstractions.

## Installation

Install only what you need. Each package pulls in `@randsum/roller` automatically.
